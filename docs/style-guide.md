# Style Guide — StudyHub

This quick reference demonstrates the visual system and example components used across the StudyHub app.

## Tokens
- --brand-1: primary brand blue used for CTAs and important accents
- --brand-2: secondary purple accent for highlights
- --bg-soft: soft page background
- --card-bg: white card surface

## Components
- AdminCard — elevated card surface for panels and list items
- AdminTable — nicely spaced table layout with hover effects and rounded header
- Buttons: `.btn-primary`, `.btn-outline` — consistent border radius and shadows
- Form controls: `.form-control` — larger hit area and polished focus state

## Pages
- Home (landing) — polished hero panel with feature tiles
- Dashboard pages — admin layout with sidebar and top nav

Generated from the active theme CSS at `src/styles/theme.css` and the admin overrides in `src/styles/admin.css`.

For live examples see `src/app/style-guide/page.tsx` in the running app.
